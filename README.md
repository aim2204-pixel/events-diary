# 📅 Мой Дневник Мероприятий

**Действующее приложение:** [https://aim2204-pixel.github.io/events-diary/](https://aim2204-pixel.github.io/events-diary/)

## 📱 О приложении
Удобный дневник для планирования мероприятий, ведения списков дел и отслеживания прогресса. 
Создавайте мероприятия, добавляйте задачи и подзадачи, отмечайте выполненное, фильтруйте по категориям и смотрите календарь.

## ✨ Возможности
- 📋 **Список мероприятий** с задачами и подзадачами
- 🏷️ **Категории** с цветовой кодировкой (Работа, Личное, Покупки и др.)
- 🔍 **Поиск** по мероприятиям и задачам
- 📅 **Календарь** с цветными точками по датам
- ✅ **Чек-листы** с прогрессом выполнения
- 📝 **Заметки** к задачам
- 🎨 **3 темы оформления** (тёмная, светлая, премиум)
- 💾 **Экспорт/импорт** данных в JSON
- 📱 **PWA** — работает офлайн, можно установить на телефон

## 🚀 Установка на телефон
1. Откройте ссылку в Chrome на Android
2. Нажмите меню (три точки) → "Добавить на главный экран"
3. Готово! Приложение открывается как родное

## 🛠️ Технологии
- Чистый HTML, CSS, JavaScript (без фреймворков)
- Font Awesome 6 для иконок
- LocalStorage для хранения данных
- Service Worker для офлайн-работы
